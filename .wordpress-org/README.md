# WordPress.org Assets

This directory contains assets for the WordPress.org plugin listing.

## Required Files

Add the following image files before submitting to WordPress.org:

### Plugin Icon
- `icon-128x128.png` - 128x128 pixels
- `icon-256x256.png` - 256x256 pixels (retina)

### Plugin Banner
- `banner-772x250.png` - 772x250 pixels
- `banner-1544x500.png` - 1544x500 pixels (retina)

### Screenshots
- `screenshot-1.png` - Dashboard overview
- `screenshot-2.png` - Webhook list
- `screenshot-3.png` - Webhook editor
- `screenshot-4.png` - Payload builder
- `screenshot-5.png` - Log viewer

## Notes

- Screenshots should match the descriptions in `readme.txt`
- Use PNG format for best quality
- Banner should represent the plugin visually
- Icon should be recognizable at small sizes
